git --version - версия гита
==========================================================
Информация для отправки вместе с проектом в репозиторий
git config --global user.name "Dmitriy"
git config --global user.email "dima_10-90@mail.ru"
==========================================================
git config --list - Просмотр информации для отправки вместе с проектом в репозиторий
--global - исп внутри одной учетной записи
--sistem - для всей учетных записей
--local - исп для конкретного проекта
==========================================================
git init - инициализация папки для отслеживания ее гитом

Содание комита
git status - файлы, которые гит не отслеживает
git add Name - добавть файл или папку Name в отслеживаемые
git add . - добавть все файлы или папки в отслеживаемые
git commit -m "Коммнетарий" - создать commit

===========================================================
Отправка
git branch -M - меняет имя главной ветки на main
git remote add origin https://github.com/DimonKZ/ballons.git - поключение к репозиторию
git remote -v - проверка подключения к гит
git push origin main - отправка
